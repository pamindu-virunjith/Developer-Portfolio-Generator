# ⚙️ DevFolio Backend

This is the backend API for the **Developer Portfolio Generator** built using Node.js and Express.

---

## 🚀 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT 

---

## 📦 Features

- RESTful API for portfolio management
- MongoDB database integration
- CRUD operations
- Error handling
- JWT authentication

---

## ⚙️ Setup Instructions

### 1. Navigate to backend folder
```bash
cd backend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Create .env file
```bash
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4. Run the server
```bash
npm start
```

---

## 📡 API Endpoints
 - POST - /api/portfolio - Create portfolio
 - GET -	/api/portfolio/:username -	Get portfolio
 - PUT -	/api/portfolio/:username -	Update portfolio
 - DELETE -	/api/portfolio/:username -	Delete portfolio
