# 🚀 DevFolio — Developer Portfolio Generator

A full-stack MERN application that allows developers to create, preview, and share a professional portfolio with a unique public URL.

---

## 🎯 Objective

Users can:
- Fill out a form with their details
- Preview their portfolio
- Publish it with a shareable URL
- Edit or delete anytime

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

---
## ⚙️ Setup (Run Locally)

### 1. Clone repo
```bash
git clone https://github.com/pamindu-virunjith/Developer-Portfolio-Generator.git
cd Developer-Portfolio-Generator
```

### 2. Setup Backend
```bash
cd backend
npm install
```
 #### Create .env file
 ```bash
 PORT=3000
 MONGO_URI=your_mongodb_uri
 JWT_SECRET=your_secret
 ```
 #### Run
 ```bash
 npm start
 ```

### 2. Setup Frontend
```bash
cd frontend
npm install
```
#### Create .env file
```bash
VITE_BACKEND_URL=http://localhost:3000
```
#### Run
```bash
npm run dev
```
