# 🎨 DevFolio Frontend

This is the frontend of the **Developer Portfolio Generator** built using React.  
Users can create, preview, edit, and view developer portfolios through a clean UI.

---

## 🚀 Tech Stack

- React.js
- React Router DOM
- Axios
- Tailwind CSS
- Framer Motion
- React Hot Toast

---

## 📦 Features

- Multi-step portfolio creation form
- Preview before publishing
- Public portfolio page (`/portfolio/:username`)
- Edit portfolio functionality
- Responsive UI design
- API integration with backend

---

## ⚙️ Setup Instructions

### 1. Navigate to frontend folder
```bash
cd frontend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Create .env file
```bash
VITE_BACKEND_URL=http://localhost:3000
```

### 4. Run the app
```bash
npm run dev
```
---

## 🌐 Routes

 - / -   Home page
 - /create -	Create portfolio
 - /preview -	Preview portfolio
 - /portfolio/:username -	Public portfolio
 - /edit/:username -	Edit portfolio

---

## 🔗 Backend Connection

### Ensure backend is running at:
```bash
http://localhost:3000
```
